# Protocol Pro Mobile App

This is the mobile app for Protocol Pro, built with React Native and Expo.

## Getting Started

### Prerequisites

- Node.js 18 or higher
- npm 10 or higher
- Expo CLI

### Installation

From the root of the monorepo:

```bash
# Install all dependencies
npm run install:all

# Or just the mobile app dependencies
npm run install:mobile
```

### Development

To start the development server:

```bash
# From the root of the monorepo
npm run mobile

# Or from this directory
npm run start
```

### Building

To build the app:

```bash
# For Android
npm run android

# For iOS
npm run ios
```

## Features

- Authentication
- Protocol management
- Task tracking
- User profiles

## Dependencies

- React Native
- Expo
- Redux Toolkit
- React Navigation
- UI components from the shared UI package
